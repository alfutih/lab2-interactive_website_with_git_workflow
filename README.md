## Requirements
- A modern web browser for example (Chrome, Firefox, Edge)
- Code editor such as VS Code

## Setup Instructions
1. Download or clone the project files.
2. Make sure all files and the “Image” folder are downloaded and they are on the same Folder.
3. To read the codes, open the project folder on some code editor, I am personal recommend you  “VS code”.
